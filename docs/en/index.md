# Don't Panic, Galvanic!

!!! info "What this guide covers"

    This guide is about building your own **galvanic electrolysis machine for permanent hair removal** — a technique where a tiny DC current through a needle destroys each hair follicle.

    **Not to be confused with:**

    - Water electrolysis (splitting H₂O into hydrogen/oxygen)
    - Metal plating / electroforming / galvanization

    You're in the right place if you want **permanent hair removal on a budget** — keep reading! 👇

## What is this?

This project is based on open-hardware designs by [scd31](https://gitlab.scd31.com/sophie/hair-electrolysis-machine-hardware) and community forks. It allows you to build a professional electrolysis machine at a fraction of the cost of commercial devices.

→ See what the device looks like in action, including a test treatment: [Video: DIY Hair Electrolysis Machine – Hand demo](https://peertube.scd31.com/w/kpZmatofZK1LyQi7SH1uo1) (PeerTube, scd31)

## Quick Start

!!! info "Only building the zoemaestra device?"
    If you only want to build the recommended zoemaestra device, you can skip any chapters or steps prefixed with **Optional:**. These sections cover alternatives (other forks, different PCB manufacturing methods), optional improvements, or background knowledge — nothing needed for the basic build.

Follow this order (don't skip!):

1. **Safety & Basics:** [Safety & Legal](safety-legal.md) (required!) → [Commercial vs. DIY](commercial-vs-diy.md)
2. **Choose & Order Hardware:** [Fork Selection](hardware-selection.md) → [PCB Manufacturing](pcb-manufacturing.md) → [Buy Components](component-sourcing.md)
3. **Finish the PCB:** [THT Soldering & Display](assembly.md)
4. **Enclosure & Pen:** [3D Printing](3d-printing.md) (case, knob, pen) → [Build the Needle Pen](pen.md)
5. **Final Assembly:** [Final Assembly](final-assembly.md)
6. **Software:** [Flash Firmware](firmware.md) → [Calibration](calibration.md) (optionally with [Multimeter Basics](multimeter-basics.md))
7. **Usage:** [First Treatment](usage.md) → [Troubleshooting](troubleshooting.md)

⭐ Optional: [Case Modding](modding.md) — silkscreen & personalize your device
